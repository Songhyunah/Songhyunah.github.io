ABOUT

Graduation Exhibition Project

구현기간 : 2019.01 ~ 2019.05

'너랑 나' 라는 청첩장 커스텀 & 판매 사이트입니다.
인생의 빛나는 순간의 시작을 함께하고 싶어서 만들었습니다.

<개발환경>
language : JSP, HTML5, CSS3, Javascript, jQuery
IDE : EditPlus

Project

프로젝트 메인 화면입니다.
주요 기능으로는 청첩장 구매, 청첩장 커스터마이징, 게시판 등이 있습니다.

청첩장을 직접 커스터마이징 할 수 있는 페이지입니다.
배경색, 패턴, 이미지 업로드, 테두리, 예식정보 등을 입력할 수 있습니다.
그리고 선택하면 바로 적용되어 나타나고,
DB에 정보가 담기게 됩니다.		 장바구니 페이지입니다.
고객이 만든 완성본 이미지 미리보기, 상품명, 수량,
용지종류, 사이즈, 금액 등이 표시되며,
총 주문금액, 장바구니 삭제, 주문하기 버튼이 있습니다.
결제 페이지입니다.
장바구니의 정보를 불러온 뒤, 배송지 정보, 결제 정보 등을 입력하게 됩니다.		 공지사항 페이지입니다.
관리자만 글을 등록할 수 있고, 글 제목, 조회수, 작성일 등이 표시됩니다.



REVIEW

소감

과 특성상 여러 가지를 배우다가 개발을 조금 배운 뒤 웹사이트 하나 전체를 구현하는 것은
부담감도 많았고, 힘들었다. 하지만 1차,2차,3차 심사 전 팀원들과 상의하고, 구글링하며
많은 것을 배웠고 부족한 점이 많다는 것도 많이 느끼게 되었다.
그래서 더 배워야겠다는 필요성을 절실히 느끼게 되기도 하고, 개발에 좀 더 관심이 생겼던 것 같다.
처음 해본 웹프로젝트였지만 졸업작품 전시회를 잘 마무리하면서 뿌듯했다.
이 프로젝트는 JSP MVC1을 활용하였는데, JSP MVC2를 활용하여 다시 만들어보고싶기도 하다.
